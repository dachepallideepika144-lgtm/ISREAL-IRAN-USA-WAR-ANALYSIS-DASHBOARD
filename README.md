Conflict Analytics Dashboard (Israel – Iran – USA)

📊 Project Overview

This project presents an interactive Conflict Analytics Dashboard built using Microsoft Excel to analyze geopolitical 
tensions between Israel, Iran, and the USA. The dashboard provides insights into military activities, attack patterns, and overall impact using structured datasets
and visual analytics.

🎯 Key Features

📌 Total Missiles Launched: 2,461
☠️ Total Deaths Reported: 1,622
📍 Dynamic filtering using slicers (Country, Date, Site Type, Attack Type)
📊 Visual representation of attack distribution and trends
✈️ Aircraft and weapon usage analysis
🕒 Timeline-based event tracking
📈 Dashboard Insights

🔹 Sites Targeted Analysis

Military locations are the most frequently targeted
City and energy sectors also show significant impact
Critical infrastructures like nuclear and oil facilities are affected
🔹 Attack Type Distribution

Iran shows higher involvement in heavy drone and missile attacks
USA and joint operations (USA/Israel) show limited but strategic strikes
🔹 Timeline Events

Conflict events are distributed across multiple dates
Indicates phased or strategic attack planning rather than continuous warfare
🔹 Aircraft Usage

Usage includes UAVs, fighter jets, and advanced bombers (B-2, B-1, B-52)
Indicates a mix of surveillance and high-impact airstrikes
🔹 Missile Trend Analysis

Initial high activity → drop → rise again
Suggests tactical pauses followed by escalation
🛠️ Tools Used

Microsoft Excel
Pivot Tables
Charts (Bar, Pie, Line)
Slicers for interactivity 📂 Dataset Description
The dataset includes:

Sites targeted
Missile and drone counts
Airstrike details
Cities affected and losses
Timeline of events
🚀 Conclusion

This dashboard highlights how data visualization can simplify complex geopolitical conflicts into understandable insights. 
It helps identify patterns in attacks, resource usage, and impact, making it useful for analytical and educational purposes.
