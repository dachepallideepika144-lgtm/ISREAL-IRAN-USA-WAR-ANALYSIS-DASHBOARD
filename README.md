📊 Conflict Analytics Dashboard (Israel – Iran – USA)

📌 Overview
This project presents an interactive **Conflict Analytics Dashboard** built using Microsoft Excel to analyze geopolitical tensions between Israel, Iran, and the USA. It visualizes attack patterns, military activities, and overall impact using structured datasets.

🎯 Key Highlights
- 🚀 **2,461** Missiles Launched  
- ☠️ **1,622** Total Deaths Reported  
- 📍 Dynamic filtering (Country, Date, Site Type, Attack Type)  
- 📊 Visual insights using charts and pivot tables  

📈 Insights
- **Military sites** are the most frequently targeted  
- **Iran** shows higher involvement in drone and missile attacks  
- **Strategic attack patterns** observed over time (phased activity)  
- Use of **UAVs, fighter jets, and bombers**  
- Missile trends indicate **tactical pauses and escalation**  

🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts (Bar, Pie, Line)  
- Slicers  

📂 Dataset Includes
- Sites targeted  
- Missile and drone counts  
- Airstrike details  
- Cities affected and losses  
- Timeline of events  

🚀 Conclusion
This dashboard simplifies complex geopolitical conflict data into clear and actionable insights. 
It helps in understanding attack trends, resource usage, and overall impact.
