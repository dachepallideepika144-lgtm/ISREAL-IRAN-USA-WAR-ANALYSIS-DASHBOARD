📊 Conflict Analytics Dashboard (Israel – Iran – USA)
📌 Overview

An interactive dashboard built in Microsoft Excel to analyze geopolitical conflict data between Israel, Iran, and the USA. It visualizes attack patterns, military activity, and overall impact using structured datasets.

🎯 Key Highlights
🚀 2,461 Missiles Launched
☠️ 1,622 Total Deaths
📍 Dynamic filtering (Country, Date, Site, Attack Type)
📊 Visual insights using charts and pivot tables
📈 Insights
Military sites are the most targeted
Iran leads in drone and missile attacks
Strategic strike patterns with phased timelines
Use of UAVs, fighter jets, and bombers
Fluctuating missile activity shows tactical escalation
🛠️ Tools Used
Microsoft Excel
Pivot Tables
Charts & Slicers
🚀 Conclusion

The dashboard simplifies complex conflict data into clear insights, helping understand attack trends, resource usage, and impact.
